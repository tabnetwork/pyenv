# pyenv
Integrated pyenv for linux and windows platform


## if linux platform, you can use python 3.8.10 after execute following command
source linux/2.2.5/setenv

pybuild 3.8.10

pyenv local 3.8.10

for more detail, please see linux/2.2.5/README.md
