# pyenv
Integrated pyenv for linux and windows platform


## if linux platform, you can use python 3.8.13 after execute following command
source linux/2.3.2/setenv

pybuild 3.8.13

pyenv local 3.8.13

for more detail, please see linux/2.3.2/README.md
