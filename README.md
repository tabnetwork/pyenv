# pyenv
Integrated pyenv for linux and windows platform


## if linux platform, you can use python 3.10.8 after execute following command
source linux/2.3.5/setenv

pybuild 3.10.8

pyenv local 3.10.8

for more detail, please see linux/2.3.5/README.md
